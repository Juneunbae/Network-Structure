OSI 참조 모델이란?
```
컴퓨터 네트워크에서 필요로 하는 기능을 7개의 계층으로 나눠서 정리한 것
```

사용 이유
```
통신에서 필요로 하는 기능이 잘 정리되어 있으므로 문제가 생긴 부분을 명확하게 알 수 있다는 장점이 있다.
```

구성 요소
```
7 계층 : 응용 계층 - 응용 프로그램 서비스 제공
6 계층 : 표현 계층 - 데이터 번역
5 계층 : 세션 게층 - 연결 관리
4 계층 : 전송 계층 - 정확한 정보 교환
3 계층 : 네트워크 계층 - 데이터 교환
2 계층 : 데이터링크 계층 - 링크의 접속
1 계층 : 물리적 계층 - 물리적 접속

하위 계층은 상위 계층에 비해 추상화된 기능을 제공하며, 상위 계층은 하위 계층을 이용하여 자체 기능을 구현한다는 사고 방식
```

```
OSI 참조 모델처럼 어떤 사고방식이나 구성으로 네트워크를 만들어낼지 여부와 같은 기본적인 체계를 '네트워크 아키텍처'라고 한다.

대표적인 네트워크 아키텍처는 OSI 모델 외 TCP/IP 모델등이 있다.
```
